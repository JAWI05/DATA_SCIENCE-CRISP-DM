Este projeto utiliza a metodologia **CRISP-DM** para analisar padrões de ausência e prever riscos de absenteísmo, fornecendo suporte estratégico baseado em dados para o departamento de Recursos Humanos.

## 📌 Destaques do Projeto

- **Metodologia CRISP-DM:** Ciclo completo de mineração de dados orientado ao negócio.
- **Tratamento de Dados:** Pipeline robusto para limpeza e imputação de dados em 18 variáveis.
- **Modelagem Avançada:** Implementação de **Random Forest** para classificação de riscos, superando modelos de regressão tradicionais em cenários de alta variabilidade.
- **Engenharia de Atributos:** Criação de scores de risco (Baixo, Médio, Alto) para tornar os resultados acionáveis.

## 📊 Principais Descobertas (Preditores)

Após a análise de importância das variáveis, os fatores que mais influenciam o absenteísmo prolongado foram:
1. **Carga de Trabalho:** Níveis elevados de demanda correlacionam-se com maior fadiga.
2. **Distância Residência-Trabalho:** Logística urbana como barreira para a assiduidade.
3. **Faixa Etária (51-60 anos):** Necessidade de monitoramento preventivo de saúde focado.

## 🛠️ Stack Tecnológica

- **Linguagem:** Python
- **Manipulação de Dados:** Pandas, NumPy
- **Machine Learning:** Scikit-Learn (Random Forest, Preprocessing)
- **Visualização:** Seaborn, Matplotlib

## 💡 Ações Estratégicas Propostas

- **Mobilidade:** Implementação de políticas flexíveis ou auxílio-transporte diferenciado para colaboradores distantes.
- **Saúde Preventiva:** Programas de bem-estar focados em grupos de risco identificados pelo modelo.
- **Gestão de Carga:** Balanceamento de tarefas para evitar o burnout e ausências reativas.
